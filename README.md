# see-a-mess

> A Vue 3 + Vite page builder project

## Requirements

- Node.js 18+
- npm 9+

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload
npm run dev

# alias for dev server
npm run start

# build for production
npm run build

# preview the production build locally
npm run preview

# run tests once
npm run test

# run tests in watch mode
npm run test:watch
```

## Stack

- Vue 3
- Vue Router 4
- Vuex 4
- Vite 5
- Sass
